CaraMenggunakanrsync

Formatumumperintahrsyncadalah:

Copyrsync[opsi]sumbertujuan

1.SinkronisasiDirektoriLokal

Copyrsync-avh/home/user/Documents//backup/Documents/

2.TransferkeServerviaSSH

Copyrsync-avz/home/user/username@server:/home/username/backup/

3.MirrorServerkeLokal

Copyrsync-avzusername@server:/var/www//home/user/backup-www/

4.BackupdenganExclude

Copyrsync-av--exclude="*.tmp"/project//backup/project/

Penjelasanopsiumum:

  • -a:archivemode(menyimpanizin,symlink,timestamp,dll)
  • -v:verbose(menampilkandetailproses)
  • -h:human-readable(ukuranfilelebihmudahdibaca)
  • -z:compress(lebihhematbandwidthsaattransfer)
