Frijal.js

Minimalist, Light-weight JavaScript Utility Library

Frijal.js adalah koleksi utilitas JavaScript yang sangat ringan dan mudah digunakan, dirancang untuk menyederhanakan tugas-tugas pengembangan umum seperti validasi data, manipulasi *array* sederhana, dan format *string*.

Instalasi

Pasang melalui NPM (Node Package Manager) atau Yarn:

npm install frijal.js

Fitur Utama

  • **isEmail()**: Validasi email cepat.
  • **safeTrim()**: Menghapus spasi tanpa error jika input null.
  • **formatCurrency()**: Format angka menjadi mata uang dengan opsi kustom.

Contoh Penggunaan

Coba fitur `safeTrim()` di konsol browser Anda:

// Input string dengan spasi berlebih
let input = "  Hello World!  ";

// Menggunakan Frijal.js (asumsi sudah di-import)
// let safeTrimmed = Frijal.safeTrim(input); 

// Dalam konteks sederhana ini, kita hanya pakai fungsi vanilla JS
let safeTrimmed = input.trim(); 

console.log(safeTrimmed); 
// Output: "Hello World!"
